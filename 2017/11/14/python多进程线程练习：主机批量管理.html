一、安装模块
paramiko模块是遵循python模块标准的openssh协议模块，可以使用pip快速安装。
首先需安装pip工具，windows的python安装包一般没有集成pip，需单独安装
下载pip源码包：
https://pypi.python.org/pypi/pip#downloads
打开windows cmd，打开后进入
解压源码包后进入源目录
执行：
python setup.py install
安装完成后，pip安装paramiko
pip install paramiko
